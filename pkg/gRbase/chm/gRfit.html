Class 'gRfit' - fitted graphical models

Description

Objects of class gRfit are created when the function fit is applied to a gModel object. When adding new types of gModel objects, one must also supply the appropritate fit function. The gRfit object contains the output of the fit which can be accessed by getFit. Separate print and summary methods exist for gRfit objects.

Usage

fit(object, ...)
getFit(x)

Arguments

object An object for which a fit method has been defined
x an object of class gRfit as created from fit applied to a gModel object.
... Additional arguments

Value

fit creates an object of class gRfit. getFit returns the fit information created by the fitting algorithm.

Examples
